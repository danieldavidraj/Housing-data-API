# Housing Data API

#### Link: [housing-data-api](https://housing-data-api.herokuapp.com/)

### Quick install guide

```
git clone https://github.com/danieldavidraj/Housing-data-API.git
```
```
cd Housing-data-API
```
```
py -3 -m venv venv
```
```
venv\Scripts\activate
```
```
pip install -r requirements.txt
```

### Endpoints

* ```GET``` /housing_data
* ```GET``` /housing_data/regions
